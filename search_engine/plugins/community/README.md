# Reviewed community plugins

Place only manually reviewed community plugins in this directory. Available
download links are listed in `../../catalogs/community_plugins.json`.

Public catalog entries are stored under `public/`, one directory per plugin.
Their `manifest.json` records the original URL, download status, size, and
SHA-256 hash. Downloaded code is not imported or executed automatically.

The current snapshot downloaded 67 of the 74 public catalog entries. Seven
upstream URLs returned HTTP 404 and remain recorded as failures in the manifest.
